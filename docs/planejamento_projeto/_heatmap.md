# Mapa de Disponibilidade (Heatmap de Horários)

<div align="justify">

O Mapa de Disponibilidade é um recurso gráfico empregado para reconhecer os períodos mais adequados para a realização de encontros em tempo real e atividades em equipe. Ele reúne os horários pessoais de todos os membros em uma única tabela de tempo (dias da semana em relação a horários).

<img src="assets/Heatmap.png" width="800"> 


Os melhores horários para as reuniões em grupo foram definidos em:

| Dia da Semana | Status de Disponibilidade | Horário |
| :--- | :--- | :--- |
| **Segunda-feira** | Alta Disponibilidade | 13h às 17h |
| **Terça-feira** | Alta Disponibilidade | 10h às 22h |
| **Quarta-feira** | Indisponível | --- |
| **Quinta-feira** | Alta Disponibilidade | 10h às 22h |
| **Sexta-feira** | Indisponível | --- |


> **Nota:** Estes dias foram selecionados com base no Mapa de Calor de horários para garantir a presença de todos os integrantes no desenvolvimento do projeto.

</div>
