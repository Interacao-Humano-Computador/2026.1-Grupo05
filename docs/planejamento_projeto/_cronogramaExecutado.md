# Cronograma de Planejamento Executado

## Introdução

O presente cronograma organiza as atividades executadas do Grupo 05 na matéria de Interação Humano Computador, para o semestre 2026.1, fundamentando-se nas diretrizes estabelecidas pelo Professor: André Barros de sales. A estrutura divide-se em: (1) Cronograma Geral executado, focado em marcos e prazos; e (2) Tabelas Específicas para cada entrega Executada.
