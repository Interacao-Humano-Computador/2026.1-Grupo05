# Ferramentas

## Introdução

As Ferramentas de Gerenciamento e execução de um projeto são partes essenciais para verificar a distribuição de tarefas entre cada membro da equipe e alinhamento entre a equipe, garantindo o cumprimento dos prazos, a execução das atividades e a rastreabilidade de cada etapa de realização do projeto.
